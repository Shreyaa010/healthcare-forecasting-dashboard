# 🏥 Healthcare Forecasting Dashboard

A Machine Learning and Data Analytics project developed using Streamlit to analyze healthcare trends and forecast future care load based on historical data.

## 📌 Project Overview

The Healthcare Forecasting Dashboard provides an interactive platform for analyzing healthcare data. Users can filter data by date, view key performance indicators (KPIs), visualize trends, and generate future predictions using a Linear Regression model.

## 🚀 Features

- 📊 Interactive KPI Dashboard
- 📅 Date Range Filtering
- 📈 Healthcare Trend Visualization
- 🔮 Future Care Load Prediction (Machine Learning)
- 📋 Dataset Preview
- 📊 Monthly Care Load Analysis
- 📥 Download Filtered Data as CSV
- 🌐 Streamlit Web Application

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## ▶️ How to Run

### Clone Repository

```bash

```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Application

